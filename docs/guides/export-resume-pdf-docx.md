# Export a ClearHire Resume as PDF or DOCX

Exporting a resume in ClearHire is the step that turns your formatted CV into a downloadable file in either PDF or DOCX format, ready to send to employers or edit elsewhere. The choice between the two is simple once you know what each is for: PDF preserves your layout exactly for applications, and DOCX stays editable for further changes.

## Steps to export

1. Open the [resume builder](/features/resume-builder) and select the resume you want.
2. Confirm your template and that the right [profile sections](/getting-started/create-your-profile) are toggled on.
3. Use the live **preview** to check page breaks, length, and spacing.
4. Choose **Export**.
5. Pick **PDF** or **DOCX**.
6. The file downloads to your device (on Android, it is saved through the native file handling).

## When to choose PDF vs DOCX

| Use PDF when | Use DOCX when |
| --- | --- |
| Applying to an employer or job portal | A recruiter asks for an editable file |
| You want the layout identical everywhere | You need to tweak wording in Word/Google Docs |
| Submitting a final, fixed version | Collaborating on the text with someone |

As a default, **export PDF to apply** and **DOCX to edit**. If a job portal does not specify, PDF is the safer choice because it cannot be reflowed by a different word processor.

## Tips for clean exports

- **Keep it to one or two pages.** Trim entries in [section control](/features/resume-builder) rather than shrinking fonts.
- **Preview before every export.** Catch awkward page breaks early.
- **Re-run the [ATS checker](/guides/ats-resume-checker)** after edits so your keyword alignment stays current.
- **Name your file clearly** after download (for example, `Firstname-Lastname-Role.pdf`).

## Troubleshooting

- **The download didn't start.** Check your browser's pop-up or download settings, then try again. On Android, confirm the app has permission to save files.
- **DOCX looks different than the preview.** Word processors substitute fonts and spacing. For a pixel-faithful result, use PDF instead.
- **The layout breaks across pages.** Reduce the number of included entries or reorder sections so content fits cleanly, then re-export.
- **A verified role isn't showing as expected.** Verification is per role; confirm the role first via [employment verification](/guides/request-employment-verification).

## FAQ

### Can I export the same resume in both formats?
Yes. Export PDF and DOCX as often as you like; there is no limit and no charge.

### Which format do most employers prefer?
PDF is the common default because it preserves layout. Use DOCX only when an editable file is requested.

### Why does my DOCX render differently elsewhere?
DOCX is editable, so the receiving word processor controls fonts and spacing. PDF avoids this by fixing the layout.

### Does exporting save the resume in ClearHire?
Your content lives in your [profile](/features/profile-management). Exporting produces a file on your device; you can regenerate the resume anytime from the same profile.

### Can I export on Android?
Yes. The export works the same way, with files saved through the Android app's native file handling.
